Bem-vindo!

Este é um parágrafo de exemplo para testar como o texto se adapta a diferentes tamanhos de tela. Observe como o tamanho da fonte e o layout mudam ao redimensionar a janela do navegador. Estamos usando a estratégia **Mobile First**, então os estilos padrão são otimizados para telas pequenas.

A ideia é começar com o que é essencial para o celular e ir adicionando complexidade e elementos visuais à medida que a tela aumenta. Isso garante uma experiência de usuário mais rápida e fluida em dispositivos móveis.

Esta imagem deve se ajustar automaticamente à largura do seu contêiner, independentemente do tamanho da tela.
